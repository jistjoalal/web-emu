# Web CPU emulator

Emulating simple CPU's for fun and learning. Inspired by [n2t](https://www.nand2tetris.org/) and [Ben Eater](https://eater.net/).

[https://jistjoalal.github.io/web-emu/](https://jistjoalal.github.io/web-emu/)
